# Playbooks and Templates

## Playbooks

- opnsense.yml: Configure firewall, VLANs, DHCP
- core_switches.yml: Set up core switch
- poe_switches.yml: Configure PoE switch
- proxmox.yml: Automate Proxmox setup

## Templates

- dhcp.xml.j2: Generate DHCP config
- vlans.xml.j2: Generate VLAN config
- firewall.xml.j2: Generate firewall rules

# Cabling and Setup

## Physical Setup

- OPNsense connected to ISP and core switch
- Core switch connected to Proxmox, TrueNAS, PoE switch
- PoE switch connected to APs
- TrueNAS connected to GPU node via iSCSI

## Cabling

- Cat6 cables for all connections
- Label all cables for easy identification
- Rack-mounted devices for neatness
